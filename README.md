# Smart Explorer

An alternative Obsidian side-pane file explorer with custom sorting, grouping, filtering, and lightweight previews.

## Features

- Sort files by name, modified date, created date, extension, or size
- Group files by folder, extension, modified month, or top-level folder
- Filter by name, extension, markdown-only, attachments-only, or recently modified
- Lightweight preview panel for Markdown, images, and other files

## Local Development

1. Clone this repository into your vault's `.obsidian/plugins/smart-explorer/` directory
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development build with watch mode:
   ```bash
   npm run dev
   ```
4. For production build:
   ```bash
   npm run build
   ```
5. Enable the plugin in Obsidian Settings → Community Plugins
